## Getting Started

Install dependencies:

```bash
yarn install
```

Run the test:

```bash
yarn test
```

Run the StoryBook:

```bash
yarn storybook
```

View Storybook dashboard in [http://localhost:6006](http://localhost:6006) in your browser

Build production app:

```bash
yarn build
```
